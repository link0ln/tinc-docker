# Tinc VPN Master Node Docker Setup

Автоматическое развертывание master ноды tinc 1.1 VPN.

## Быстрый старт

```bash
docker-compose up -d
```

Вот и все! Master нода автоматически:
- Сгенерирует RSA и Ed25519 ключи
- Настроит сеть 10.200.210.0/24
- Назначит себе IP 10.200.210.1
- Откроет порт 655 для входящих соединений

## Конфигурация

- **Имя сети**: gnet
- **Имя master ноды**: master
- **IP адрес**: 10.200.210.1/32
- **Сеть**: 10.200.210.0/24
- **Порт**: 655 (TCP/UDP)
- **Интерфейс**: tun0

## Добавление клиентов

```bash
./add-client.sh client1
```

Скрипт выдаст invitation URL, который нужно передать клиенту.

## Структура

- `tinc-data/` - автоматически создаваемая папка с конфигурацией и ключами
- `init-tinc.sh` - скрипт автоматической инициализации
- `add-client.sh` - скрипт добавления новых клиентов

## Proxy wrapper

Дополнительно запускается 3proxy для проксирования трафика:
- SOCKS5: порт 5736
- HTTP proxy: порт 5737
